Demo manga pages for the "The Black Death: A Medieval Pandemic" lesson.

Save your three manga page images in THIS folder with these exact names:

  black-death-1.png   <- page 1 (1347 A.D. Messina, ships arrive)
  black-death-2.png   <- page 2 (the plague spreads, the master falls ill)
  black-death-3.png   <- page 3 (the aftermath and rebuilding)

The book reader shows page 1 on the right of the first spread, then pages 2 and
3 on the next spread. If a file is missing or fails to load, the reader falls
back to the structured text panels automatically — nothing breaks.

PNG is assumed by the filenames above. To use .jpg instead, change the paths in
lib/demo-pages.js to match.
